# 📊 Instagram User Engagement Analysis (SQL)

## 🚀 Project Overview

This project analyzes user engagement patterns on an Instagram-like platform using SQL to derive actionable insights and support business decisions.

---

## 🎯 Objectives

* Perform data cleaning (handle NULLs and duplicates)
* Analyze user activity (posts, likes, comments)
* Segment users based on engagement levels
* Generate data-driven business insights

---

## 🛠️ Tools Used

* SQL (MySQL)

---

## 📂 Dataset

The dataset includes:

* Users
* Photos
* Likes
* Comments
* Follows
* Tags

---

## 🔍 Key Analysis

### Data Cleaning

* Checked for NULL values
* Removed duplicate records

### User Engagement Analysis

* Calculated total posts, likes, and comments per user
* Used joins across multiple tables

### User Segmentation

* Inactive users
* Low activity users
* Medium activity users
* High activity users

---

## 📈 Key Insights

* 77% users are highly active
* 16% users are low or inactive
* Very few users fall in the medium engagement category

---

## 💡 Business Recommendations

* Improve user retention strategies
* Encourage content creation through incentives
* Personalize content to increase engagement
